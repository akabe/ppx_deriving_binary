Module Ppx_deriving_binary_bytes_runtime.GreedyList

Greedy lists

'a GreedyList.t is a type of lists whose elements have a type 'a. Its elements are decoded as many as possible.

For example, binary data

00  11  22  33  44  55  66  77    88  99  aa  bb  cc  dd
<--- x[0] --->  <--- x[1] --->    <--- x[2] --->

can be decoded to a list [0x00112233l; 0x44556677l; 0x8899aabb] as a type uint32be GreedyList.t. The trailing cc dd is not able to converted due to lack of length.

type 'a t = 'a list
val of_binary_bytes : ('a -> 'b -> 'c * 'b) -> 'a -> 'b -> 'c list * 'b
val binary_bytes_of : ('a -> 'b -> unit) -> 'a -> 'b list -> unit
